# HezzlBot v7 — Autopilot Farmer

Автоматический бот для игры **Hezzlmania** (Билайн).

## Что умеет

- 🎮 Автоматически играет в Match3 (получает жизни → играет → собирает монеты)
- 💌 Рассылает жизни по всем 259 кланам сервера
- 📥 Автоматически принимает входящие жизни от клансеров
- 🛒 Покупает жизни за монеты когда они заканчиваются
- 🎁 Автоматически забирает все доступные награды (Seasonbox, my-awards)
- 🎲 Играет в Cubes Master когда есть токены
- 🔑 **Автоматически обновляет токен каждые 20ч** (не нужно перехватывать каждый день)
- 🔄 Батник автоматически перезапускает бот при любом сбое

## Структура проекта

```
bot_bundle/
├── bot_v7.js           # Основной бот
├── запустить_бот.bat   # Запуск (двойной клик)
├── mitm_capture.py     # Перехват токена (mitmproxy)
├── package.json        # Зависимости Node.js
└── logs/
    └── latest_token.txt  # Токен (создать вручную, не в git!)
```

## Быстрый старт

### 1. Установка

```bash
npm install
```

### 2. Получение токена (первый раз)

Токен нужен **только один раз** — потом бот сам его обновляет каждые 20 часов.

**Способ: mitmproxy (перехват трафика)**

1. Установить [mitmproxy](https://mitmproxy.org/)
2. Настроить прокси на телефоне (порт 8080)
3. Запустить: `mitmproxy -s mitm_capture.py`
4. Открыть Мой Билайн → перейти в игру
5. Токен автоматически сохранится в `logs/latest_token.txt`

### 3. Запуск бота

Двойной клик на `запустить_бот.bat`

Или из консоли:
```bash
node bot_v7.js
```

## Токен

- Токен — это **JWE** (зашифрованный JWT), выдаётся Билайном при входе в игру
- Живёт **24 часа**
- Бот обновляет его сам каждые 20ч через `/auth/v1/game/{id}/refresh-token`
- При первом запуске или после перезапуска через 24ч — нужно обновить вручную
